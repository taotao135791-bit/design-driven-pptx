## Rules

- Chapter consistency: ALL chapters must have a [chapter] transition page, or NONE.
- TOC consistency: If [table_of_contents] exists, chapter list must match all [chapter] titles.
- Content fidelity: Strictly follow the source document. No additions, no deletions.
- One core argument per content page.
