# General

> Applicable scenarios: General presentation scenarios that do not clearly match other specialized categories, such as meeting sharing, project introduction, work communication, internal discussions, etc.
> Style anchor: Professional, clear, moderately polished general business style

## Design Philosophy

- **Content-rich**: Pages must not be empty; every page should have substantive content, avoiding large blank areas or sparse information
- **Professional restraint**: Neither over-decorated nor overly plain, maintaining appropriate aesthetics and professionalism
- **Clear structure**: Distinct logical hierarchy, well-organized information, enabling readers to quickly grasp key points
- **Rich visuals**: Make good use of charts and SmartArt to aid information delivery, avoiding walls of pure text

## Information Density: Medium-High

- Content area fill rate **65-80%**, pages should look substantial but not crowded
- Each page carries one complete topic, providing sufficient arguments and data support around that topic
- Encourage chart+text combinations to present multi-layered information on a single page
- Avoid sparse pages with a single large image filling the entire page or only 2-3 bullets

> **Principle: After reading a page, the reader should feel "the information is complete and clearly expressed" — neither hollow nor overwhelming**

## Text-to-Visual Ratio: Balanced

- Recommended ratio for content pages: approximately **50% text** + **35% charts/SmartArt** + **15% whitespace**
- Encourage using charts (bar charts, line charts, pie charts, etc.) to visualize data for more intuitive presentation
- Encourage using SmartArt to express processes, relationships, hierarchies, and other logical structures
- Avoid text-only pages — if a page contains only text, consider whether some content can be replaced with charts or SmartArt

## Color Guidance

- If the user provides brand colors, prioritize using brand colors
- Recommend a **three-color system**: primary color + grayscale + one accent color, maintaining visual consistency
- Light colors preferred; for special themes like technology, use scene-appropriate dark color schemes + white text
- Strictly avoid overly flashy multi-color combinations; use no more than 4 colors across the entire document (including grayscale)

## Font Guidance

- Titles: Sans-serif Bold, ensuring visual hierarchy prominence
- Body text: Sans-serif Regular, maintaining reading comfort
- Font size hierarchy: Cover title **40-52px** → Page title **26-32px** → Subtitle **22-26px** → Body text **18-22px** (use 22px when page text content is light, 20px for moderate, 18px for heavy; must not go below 18px) → Footnotes **12-16px**

## Content Pages

- Recommended layout: **Page title** (top) → **Thin gray divider line** → **Free content area** → **Page number** (bottom-right)
- Page title should be a summary sentence of the page's core point; reading the title conveys the page's main message
- Use SmartArt moderately to build flowcharts, relationship diagrams, and hierarchy diagrams

## Narrative Style: Logically Clear

- Each page title summarizes the core point, rather than being a simple topic word
- Information organized by logical hierarchy: general → specific, cause → effect, problem → solution
- Language is concise and professional, data-driven, avoiding vague descriptions
- Key conclusions and data are bolded for quick scanning

## Content Expression Techniques

- **Charts**: Bar charts, line charts, pie charts, and other data visualizations to make data more persuasive
- **SmartArt**: Flowcharts, hierarchy diagrams, cycle diagrams to express logical relationships, more intuitive than plain text (SmartArt is implemented in PPTD through shape + text + line combinations)
- **Tables**: Multi-dimensional comparisons, data summaries, suitable for structured information display
- **KPI Big Numbers**: Key metrics prominently displayed to attract attention
- **Structured Lists**: Each bullet group has 3-5 items, containing specific data rather than vague descriptions
- **Emphasis Marking**: Key data and conclusions bolded for quick reader takeaway

## Decoration Prohibitions

| Prohibited | Alternative |
|--------|---------|
| Over-decoration/flashy effects | Clean, professional visual design |
| Large blank areas | Substantial content filling |
| Text-only pages | Text-visual combinations with charts or SmartArt |
| Too many colors (>4) | Three-color system: primary + grayscale + accent |
| Decorative icon overload | Use icons only when they aid comprehension |
| Gradient and shadow abuse | Flat solid colors as the primary approach |
| Rounded rectangles | Unless specifically required by the user, prefer sharp-cornered rectangles or clean geometric divisions to maintain a professionally restrained visual style |
